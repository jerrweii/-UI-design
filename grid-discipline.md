# Grid Discipline

## Purpose

Use grids to make layouts predictable, aligned, and easier for developers to implement.

## Repeatable Practice

- Use a column grid for page and section structure.
- Use a baseline or spacing grid for vertical rhythm.
- Prefer an `8px` spacing system for margins, gaps, padding, and common dimensions.
- Use `4px` only for fine details.
- For web layouts, consider a 12-column grid when the content requires flexible sections.
- Keep gutters and side margins consistent within each breakpoint.

## Development Checklist

- Layout values map to spacing tokens.
- Columns, gutters, and max widths are explicit.
- Components align to the same system.
- No arbitrary one-off offsets are introduced without a reason.
