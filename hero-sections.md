# Hero Sections

## Purpose

Use hero sections to establish value, trust, and the primary action quickly.

## Repeatable Practice

- Make the headline immediately understandable.
- Avoid jargon in the main value statement.
- Include supporting copy that clarifies the offer.
- Show trust cues when relevant.
- Make the primary CTA visually distinct.
- Use secondary CTAs carefully so they do not compete with the main action.
- Use visual impact without misleading or overwhelming the user.

## Development Checklist

- The first viewport communicates what the product or page is.
- The main CTA is visible and action-oriented.
- Hero media supports the message.
- Text remains readable across viewport sizes.
- Secondary actions are visually lower priority.
